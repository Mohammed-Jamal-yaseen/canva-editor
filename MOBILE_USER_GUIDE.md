### Troubleshooting

#### Can't See Bottom Toolbar
- The toolbar is fixed at the bottom
- Scroll up if content is covering it
- Check if you're in fullscreen mode

#### Grid View Not Showing
- Zoom out more (pinch in)
- Grid view activates at zoom < 25%
- Or use the grid toggle button

#### Page Not Responding
- Try tapping outside the canvas
- Close any open sidebars
- Refresh the page if needed

#### Zoom Controls Not Working
- Make sure you're not in grid view
- Try using pinch gestures instead
- Check if an object is selected

### Keyboard Shortcuts (External Keyboard)

- **Ctrl/Cmd + Z** - Undo
- **Ctrl/Cmd + Y** - Redo
- **Ctrl/Cmd + C** - Copy
- **Ctrl/Cmd + V** - Paste
- **Delete** - Delete selected object
- **Ctrl/Cmd + A** - Select all

### Performance Tips

#### For Smooth Experience
1. **Close unused apps** to free memory
2. **Use WiFi** for faster image loading
3. **Clear browser cache** if slow
4. **Restart app** if experiencing issues
5. **Update browser** to latest version

#### Recommended Browsers
- **Chrome** (Android)
- **Safari** (iOS)
- **Edge** (Windows)
- **Firefox** (All platforms)

### Getting Help

#### Common Issues
- **Slow performance** - Reduce number of objects
- **Can't add text** - Make sure text tool is selected
- **Image won't upload** - Check file size and format
- **Changes not saving** - Check internet connection
